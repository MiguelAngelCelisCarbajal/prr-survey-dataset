# Parametric Roll Resonance (PRR) - Systematic Literature Review Dataset

This repository contains the complete dataset and classification for the research paper:  
**"Parametric Roll Resonance in Ships: A Survey of Physical-Based, Data-Driven, and Hybrid Forecasting Approaches"**

## 📌 Overview
This systematic mapping study analyzed a total of **325 documents**. While the main manuscript discusses a curated selection of 133 key references to ensure a focused technical discussion, this repository provides the full taxonomy and classification for all identified records. This ensures transparency and supports research reproducibility in maritime engineering and AI-based stability forecasting.

## 📂 Repository Contents
- **`PRR_Full_Database.xlsx`**: Complete list of 325 papers classified by Modeling Paradigm (F), Data Source (S), and Maturity Level (L).
- **`Full_Library.bib`**: Complete BibTeX file containing all 325 analyzed works for easy integration into LaTeX projects.

## 📊 Classification Taxonomy Summary
The papers are categorized based on the following criteria defined in the study:

* **L1–L6 (Algorithmic Maturity):** Ranging from classical statistical models (L1) to advanced self-adaptive systems (L6).
* **S1–S5 (Data Source Fidelity):** From analytical equations (S1) to high-fidelity CFD (S3) and full-scale onboard sensor data (S5).
* **F (Modeling Paradigms):** Deterministic, Stochastic, Machine Learning, Deep Learning, and Hybrid/Physics-Informed models.
